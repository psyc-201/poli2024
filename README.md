**PSYCH 201A: Simulation Project [FALL 2025]**

*This repository contains the code and documentation for a reproduction project in the PSYC 201A course led by Dr. Bria Long at UC San Diego. The goal of this project is to perform a partial reproduction of the findings from Poli et al. 2024.*

Required dependencies: 

python                    3.10.19
pymc                      5.25.1
numpy                     2.2.6
pytensor                  2.31.7
scipy                     1.15.2
